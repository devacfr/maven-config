## Get it now

To use Maven Config in your Maven project site, add it to pom.xml:

```xml
<parent>
    <groupId>org.cfr</groupId>
    <artifactId>maven-config</artifactId>
    <version>5</version>
</parent>
```

[Full usage instructions &raquo;][maven-config-usage]
